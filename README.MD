# How to run the website locally using node.js:

1. Make sure that nodejs an npm are installed:\
`$ sudo pacman -S nodejs`\
`$ sudo pacman -S npm`

2. Install the http-server package with npm:\
`$ sudo npm install -g http-server`

3. Run http-server:\
`$ http-server`
